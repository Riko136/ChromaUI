The application should provide a graphical user interface for CRUD operations on a vector database.

The vectors should also be displayed.

The metadata should also be displayed.

Queries should be possible based on text input. The vectors should be displayed for the text input.

Multiple results for a given text input should be displayed and sorted by the vectors.

Alternatively, queries based on metadata should also be possible.

 

Chroma-DB should be used as the vector database.

Node.js and React should be used as the programming environment.

- User and Database Management
- Collection Forking
- File uploads
- Persistant collection selection and item detail panel sizing
- More colors?
- 2D projection (UMAP/PCA) scatter plot of embeddings?

- Collection configuration
